## 📁 Repository Structure

This repository contains **17 modules** covering Python fundamentals through machine learning. Each folder contains a Jupyter Notebook with daily tasks and a README summarizing key learnings.

| # | Module | Topics Covered |
|---|--------|----------------|
| 01 | [Python Introduction](./01.Python-Introduction/) | Syntax, variables, data types, operators |
| 02 | [Data Structure](./02.Data-Structure/) | List, tuple, set, dictionary |
| 03 | [Function and Methods](./03.Function-and-Methods/) | Functions, lambda, built-in methods |
| 04 | [Logic, Control Flow & Looping](./04.Logic-Control-Flow-and-Looping/) | If/else, for loop, while loop, comprehension |
| 05 | [Quantitative Data Analysis](./05.Quantitative-Data-Analysis/) | Descriptive statistics, NumPy |
| 06 | [Pandas Introduction](./06.Pandas-Introduction/) | DataFrame, Series, basic operations |
| 07 | [Akses, Indexing & Transformasi Data](./07.Akses-Indexing-dan-Transformasi-Data/) | loc, iloc, apply, map, transform |
| 08 | [Data Wrangling](./08.Data-Wrangling/) | Cleaning, handling missing values |
| 09 | [Merging & GroupBy](./09.Merging-dan-GroupBy/) | merge, concat, join, groupby, agg |
| 10 | [Time Series](./10.Time-Series-Rev/) | Datetime indexing, resampling, rolling |
| 11 | [Matplotlib Introduction](./11.Matplotlib-Introduction/) | Line, bar, scatter, histogram |
| 12 | [Customizing Visualization](./12.Customizing-Visualization/) | Styling, colors, annotations, themes |
| 13 | [Multiple Subplots](./13.Multiple-Subplots/) | subplot, GridSpec, layout management |
| 14 | [Quantitative Comparison](./14.Quantitative-Comparison/) | Comparative charts, dual-axis, heatmaps |
| 15 | [Seaborn](./15.Seaborn/) | Statistical plots, pairplot, heatmap, violin |
| 16 | [Data Preprocessing](./16.Tipe-Data-Handling-Missing-Outliers-Scaling/) | Data types, missing values, outliers, scaling |
| 17 | [Machine Learning — Clustering](./17.Machine-Learning-Unsupervised-Clustering/) | K-Means, DBSCAN, PCA, elbow method |
