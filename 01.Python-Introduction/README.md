## 🧠 What I Learned

This module introduced the foundational building blocks of Python — from basic syntax
and arithmetic operations to variables and core data types. While these concepts may
seem simple, having a solid grasp of how Python handles strings, integers, and floats
is essential before working with data libraries like Pandas or NumPy.

---

## 📌 Topics Covered

- `print()` function & basic syntax
- Arithmetic operations in Python
- Variables — integer & float data types
- String operations
- Data type overview

## 💡 Key Takeaways

Python's dynamic typing makes it flexible, but understanding the difference between
data types (int, float, string) early on prevents a lot of bugs later — especially
when cleaning and transforming real datasets.
